## Bot Messaging Spec
I want the user experience to be straightforward. The notification system should send the user a message that they have recieved a message. This notifcation should show up in the application (Game-World) and in Discord itself. This will allow the user to see notifications whether they are inside of the app or outside of it. I also want the user to be able to join communities, create communities, delete communities (ONLY IF THEY ARE THE CREATOR OF SAID COMMUNITY. TO ENSURE THAT OTHER USERS CAN'T DESTROY OTHER USERS WORK.), and finally also have the option to leave (Since they were able to join as well.) I want this user interaction to be similar to Discord's integrity.
